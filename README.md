# Infrastructure/Detection as Code | CI/CD Pipelines
## Overview

This repository implements Infrastructure as Code (IaC) and Detection as Code (DaC) Pipelines that automatically deploy Azure resources and Sentinel analytics.

## Repository Structure
### detection-as-code-CI-CD-pipeline/.github/workflows

YAML files used to push resources to Azure.

### detection-as-code-CI/CD-pipeline/detections

ARM templates of Sentinel analytics.

## Goals

- Demonstrate end-to-end deployment of resources.
- Automate the deployment of resources.
- Make deploying resources as stream lined as possible.

## Notes

This repository is actively maintained and updated with new piplines, improvements, and IaC/DaC projects over time.
